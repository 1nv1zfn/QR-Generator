# 🎨 QR-Generator

**Генератор стильных QR-кодов с логотипом**


## 📌 О проекте

**QR-Generator** — это простая программа для создания красивых QR-кодов с возможностью вставки логотипа в центр. 

Программа работает полностью **офлайн**, не требует установки и подходит для всех, кому нужны стильные QR-коды для бизнеса, соцсетей или личных проектов.

---

## 🚀 Возможности

-  Создание QR-кодов из текста или ссылки
-  Вставка логотипа в центр QR-кода (любое изображение)
-  Выбор цвета точек (8 цветов + свой цвет)
-  Сохранение результата в PNG с высоким качеством
-  Работает полностью офлайн (без интернета)
-  Один EXE-файл (никаких установок)
